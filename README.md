# 🍽️ Culinary Dispatch System

A modern, full-stack food delivery platform that connects hungry customers with their favorite local restaurants. Built with React, Node.js, Express, and MySQL.

## ✨ Features

### 🔐 User Management
- Secure customer registration and authentication
- Role-based access (Customer/Admin)
- JWT-based authentication
- Profile management

### 🍕 Restaurant & Menu
- Browse restaurants by cuisine type
- Extensive menu management
- Real-time menu item availability
- Rich item descriptions and pricing

### 🛒 Order Management
- Intuitive cart functionality
- Real-time order tracking
- Multiple delivery addresses
- Order history

### 👑 Admin Dashboard
- Restaurant management (Add/Update/Delete)
- Menu item control
- Order monitoring
- Customer management

## 🛠️ Tech Stack

- **Frontend**: React, TailwindCSS
- **Backend**: Node.js, Express
- **Database**: MySQL, Sequelize ORM
- **Authentication**: JWT
- **Admin Interface**: Python/Tkinter

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/JeevanKumar-u/food-delivery-system_DBMS.git
```

2. Install dependencies
```bash
# Backend dependencies
cd src
npm install

# Frontend dependencies
cd react/client
npm install
```

3. Set up environment variables
```bash
# Create .env file in root directory
DB_HOST=localhost
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=culinary_dispatch_system
JWT_SECRET=your_jwt_secret
```

4. Start the application
```bash
# Start backend server
npm run start

# Start frontend application
cd react/client
npm start
```
